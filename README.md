# Practiques de Laboratori — Quimica Analitica

Pagina web interactiva per als estudiants de practiques de laboratori de Quimica Analitica, Grau de Farmacia, Universitat de Barcelona.

**[Obrir la pagina](https://gomezsancheza.github.io/practiques-laboratori/)**

## Que conte

- **30 practiques** interactives (S01-S09 estandaritzacions + D01-D17c determinacions), cadascuna amb:
  - Resum de la reaccio i l'indicador
  - Procediment pas a pas
  - Simulador de valoracio interactiu
  - Consells per evitar errors comuns
- **3 seminaris** (pesada per diferencia, mesura de volums, la llibreta de laboratori)
- **Index** amb navegacio lateral i cerca

## Tecnologia

HTML + CSS + JS pur, sense frameworks ni build step. Desplegat directament amb GitHub Pages.

| Arxiu | Funcio |
|-------|--------|
| `shared.css` | Estils compartits per a les 30 practiques |
| `shared.js` | Sistema de pestanyes i helpers del simulador |
| `seminari.css` | Estils per als 3 seminaris |
| `index.css` | Estils de la pagina d'index |

Les formules matematiques es renderitzen amb [MathJax 3](https://www.mathjax.org/).
